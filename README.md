# Let's Flow — Site web officiel

## Images

Toutes les images sont en `.webp` (optimisées) dans le dossier `images/`.

## SEO & GEO

- Balises Open Graph et Twitter Card sur toutes les pages
- Schema.org JSON-LD (WebPage + FAQPage) sur toutes les pages
- `sitemap.xml` et `robots.txt` inclus
- `llms.txt` pour l'indexation par les IA

## Déploiement

Pousser sur la branche `main` → GitHub Pages publie automatiquement.
Utiliser le fichier `pousser-sur-github.bat` pour pousser depuis Windows.
